Question a)

Consider the cars data set. Assume dist is the response variable and speed is the predictor variable. Create a linear model using the lm() function (R) or an equivalent in Python.

R (using lm())

model <- lm(dist ~ speed, data = cars)
      summary(model)

Python (using statsmodels)

import statsmodels.formula.api as smf

      model = smf.ols('dist ~ speed', data=cars).fit()
      print(model.summary())
