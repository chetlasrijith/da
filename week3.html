Week 3 - Programs using Statistics in Python

This section demonstrates the implementation of basic statistical concepts like Mean and Median using different Python libraries.

1. Mean (Average)

Theory:
Mean is the average of a dataset. Formula: Mean = (Sum of all values) / (Number of values)

Using NumPy


import numpy as np

data = [10, 20, 30, 40, 50]
mean_value = np.mean(data)

print("Mean using NumPy:", mean_value)

Output: 30.0

Using Python Built-in sum() and len()


data = [10, 20, 30, 40, 50]

mean_value = sum(data) / len(data)

print("Mean using built-in functions:", mean_value)

Output: 30.0

Using Pandas


import pandas as pd

data = pd.Series([10, 20, 30, 40, 50])

mean_value = data.mean()

print("Mean using Pandas:", mean_value)

Output: 30.0

Using Statistics Module


import statistics

data = [10, 20, 30, 40, 50]

mean_value = statistics.mean(data)

print("Mean using statistics module:", mean_value)

Output: 30

2. Median

Theory:
Median is the middle value in a sorted dataset. If the number of values is even, median is the average of the two middle numbers.

Using NumPy


import numpy as np

data = [10, 20, 30, 40, 50]

median_value = np.median(data)

print("Median using NumPy:", median_value)

Output: 30.0

Using Pandas


import pandas as pd

data = [10, 20, 30, 40, 50]

df = pd.Series(data)

median_value = df.median()

print("Median using Pandas:", median_value)

Output: 30.0

Using Statistics Module


import statistics

data = [10, 20, 30, 40, 50]

median_value = statistics.median(data)

print("Median using statistics module:", median_value)

Output: 30

Using Python Built-in Method


def calculate_median(data):
    sorted_data = sorted(data)
    n = len(sorted_data)
    mid = n // 2

    if n % 2 == 0:
        return (sorted_data[mid - 1] + sorted_data[mid]) / 2
    else:
        return sorted_data[mid]

data = [10, 20, 30, 40, 50]

median_value = calculate_median(data)

print("Median using built-in method:", median_value)

Output: 30
