Aim: To install and load data analytics related packages in R.

EXPERIMENT 9: Using as Operator

Aim: To convert data types.

Aim: To create matrix.

Aim: To work with factors.

EXPERIMENT 13: Lists

EXPERIMENT 14: Aggregation using aggregate()

Aim: To find group-wise averages.

aggregate(Height ~ Gender, data=data, mean)
aggregate(Weight ~ Gender, data=data, mean)

EXPERIMENT 15: Group Processing using ddply()

Aim: To perform group-wise processing.

ddply(data, .(Gender), summarise,
 Avg_Height = mean(Height),
 Avg_Weight = mean(Weight))

EXPERIMENT 16: Invalid Values and Outliers

Aim: To detect invalid values and outliers.

anyNA(data)
boxplot(data$Height, main="Outliers in Height")
boxplot(data$Weight, main="Outliers in Weight")

Aim: To identify problems in data using visualization.

hist(data$Weight, main="Histogram of Weight", col="lightgreen")
plot(data$Height, data$Weight,
 main="Height vs Weight",
 xlab="Height", ylab="Weight")
